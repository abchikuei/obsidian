### 🧭 Navigation

| Shortcut                   | Action                              |
| -------------------------- | ----------------------------------- |
| `⌘ + K`                  | Quick switch to any chat or contact |
| `⌘ + F`                  | Search in current chat              |
| `⌘ + ▲ / ▼`          | Move to next / previous chat        |
| `⌘ + ⌥ + ▲ / ▼` | Move between unread chats           |
| `⌘ + 0`                  | Jump to Saved Messages              |
| `⌘ + 1-5`                | Jump to pinned chat by number       |
| `⌃ + Tab`            | Switch to next chat                 |
| `⌃ + ⇧ + Tab`    | Switch to previous chat             |
| `⌥ + ▲ / ▼`       | Switch between chats                |

### ✉️ Messaging

| Shortcut    | Action                                     |
| ----------- | ------------------------------------------ |
| `⌘ + R`     | Record voice message                       |
| `⌘ + Enter` | Send message                               |
| `⇧ + Enter` | New line without sending                   |
| `▲`         | Edit last sent message when input is empty |

### 📝 Text Formatting

| Shortcut | Action |
| -------- | ------ |
| `⌘ + B` | Bold |
| `⌘ + I` | Italic |
| `⌘ + U` | Underline |
| `⌘ + ⇧ + X` | Strikethrough |
| `⌘ + ⇧ + M` | Monospace / code |
| `⌘ + ⇧ + P` | Spoiler |
| `⌘ + K` | Create a link |
| `⌘ + ⇧ + N` | Remove formatting, plain text |

### 🪟 Window

| Shortcut | Action |
| -------- | ------ |
| `⌘ + L` | Lock Telegram |
| `⌘ + M` | Minimize Telegram |
| `⌘ + Q` | Quit Telegram |
| `⌘ + ,` | Open settings |
| `⌘ + J` | Show contacts |
