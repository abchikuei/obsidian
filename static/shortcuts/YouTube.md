### ⏯️ Playback

| Shortcut       | Action                  |
| -------------- | ----------------------- |
| `K` or `Space` | Play / pause            |
| `M`            | Mute / unmute           |
| `⇧ + <`    | Decrease playback speed |
| `⇧ + >`    | Increase playback speed |

### ⏩ Seeking

| Shortcut | Action |
| -------- | ------ |
| `J` | Rewind 10 seconds |
| `L` | Skip forward 10 seconds |
| `0-9` | Jump to 0%–90% of the video |

### 📺 Chapters & Playlist

| Shortcut | Action |
| -------- | ------ |
| `⇧ + P` | Previous video |
| `⇧ + N` | Next video |

### 🖼️ View

| Shortcut | Action |
| -------- | ------ |
| `F` | Toggle fullscreen |
| `T` | Toggle theater mode |
| `I` | Toggle miniplayer |
| `/` | Go to search bar |
| `Esc` | Close miniplayer or current dialog |

### 💬 Subtitles

| Shortcut | Action |
| -------- | ------ |
| `C` | Toggle captions on / off |
| `O` | Cycle caption text opacity levels |

### 🧰 Other

| Shortcut | Action |
| -------- | ------ |
| `⇧ + ?` | Show all YouTube keyboard shortcuts |
