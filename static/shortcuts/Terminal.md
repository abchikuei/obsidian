## 🐚 Ghostty

| Shortcut      | Action             |
| ------------- | ------------------ |
| `⌘ + D`     | Split the terminal |
| `⌘ + K`     | Clear the output   |
| `⌘ + <n>`   | Go to the nth tab  |
| `⌘ + Enter` | Toggle fullscreen  |

## 🗂️ Yazi

| Shortcut | Action                                                             |
| -------- | ------------------------------------------------------------------ |
| `c`      | Copy, opens a submenu like `cc` for full path or `cf` for filename |
| `z`      | Jump, uses zoxide to move to frequent folders                      |
| `g`      | Go to prefix                                                       |
| `r`      | Rename, opens an input bar                                         |
| `f`      | Filter, hide files that do not match                               |
| `v`      | Visual mode, select multiple files                                 |
| `y`      | Yank, copy a file to the clipboard                                 |
| `p`      | Paste                                                              |
| `d`      | Delete, moves the selected item to the Trash                       |
| `Tab`    | Show the size, timestamps of folder or file                        |
| `.`      | Show hidden files and folders                                      |
